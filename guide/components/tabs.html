# Tabs 标签页

Tab 展示了与当前主页面视图相关的子视图，使得用户能够轻松地在当前页面中切换不同的子视图来查看内容。

使用原则

  • 同一组 Tab 的样式与形式需要保持一致。
  • 按钮文字与按钮图标的颜色在各种情况下都保持一致。
  • Tab 上的文字需统一，并且不可省略。
  • 有一个 Tab 始终被默认选中。

基本用法

Tabs 组件默认选中第一个标签页，你也可以通过 selected 属性来指定当前选中的标签页。
