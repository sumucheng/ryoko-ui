# Select 选择器

允许用户从一列选项中选择一个或多个，并呈现最后的选择结果的一种选择器控件。

使用原则

  • 当选项超过20个以上并且存在动态增加时，应当加入搜索功能。未超过的则使用无搜索的形式，建议一次最多展示5条信息，多余的靠上下滑动来查阅。
  • 已选择项需在顶部集中展示。
  • 当再次打开下拉选择时，需明确的区分已选项和未选项，并且与 hover 的状态也需要一同区分开。

基础用法

适用广泛的基础单选和多选。
