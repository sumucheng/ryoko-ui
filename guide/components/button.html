# Button 按钮

按钮用于触发操作。

使用原则

  • 按钮文字与按钮图标的颜色在各种情况下都保持一致。
  • 每个页面通常只有不超过三个的主要按钮，给用户明确的操作指引。
  • 加载状态时，加载指示器替代按钮文字，不影响按钮的尺寸与位置。

基础按钮

主要按钮承载的是用户优先级最高的按钮，给用户明确的操作指引，帮助用户完成操作。
