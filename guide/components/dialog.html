# Dialog 对话框

在保留当前页面状态的情况下，告知用户并承载相关操作。

使用原则

  • 对话框是用于在不离开主路径的情况下，提供用户快速执行简单的操作、确认用户信息或反馈提示的辅助窗口。

基础弹窗

基础的模态弹窗由标题、内容区和操作区构成
